Venue Host 0.3.0.0

- Added Event Shouts tab for reusable venue-wide custom shout macros.
- Added Event Shout window as the global automatic shout enclosure.
- Added per-shout Start/End windows so each custom shout can run on its own schedule.
- Added manual Shout Now action for selected custom shouts.
- Added non-blocking Add/Edit Event Shout editor window with macro preview.
- Added FIFO queue spacing for simultaneous Event Shouts so due announcements stagger cleanly.
- Projected Event Shout queue preview now shows expected staggered send times.
- Added Event Shout macro variables for shout names and shout window times.
- Kept the service-based architecture: EventShoutService owns timing, queue state, preview, and send behavior.
- Polished Event Shouts tables and controls to match the existing DJ Lineup and Staff Schedule UI.
